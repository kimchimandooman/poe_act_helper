# poe_act_helper
#Requires AutoHotkey v2.0
https://www.autohotkey.com/

====주의사항====


1.fhd 해상도를 사용하고있어서 타 해상도 사용시 오토핫키 설치시 같이 설치되는 autohotkey window spy 를 사용해서 직접 범위를 지정해야 합니다. 


2.기본 폰트가 아닌 다른 폰트를 사용하고 있기때문에 숫자가 들어가는 부분은 해당 부분의 인게임 스샷을 직접 찍어서 사용해야합니다. 사진들 올려져있는거 보고 참고 하세요 


hp는 해당 부분에 검은색을 찾고 찾으면 1번을 눌러주는 기능입니다.
